**Vlad Installed Applications**

Vlad can install the following technologies:

- Apache2
- Adminer
- [Drupal Install Scripts](drupal.md)
- [Drush](drupal.md) (with the 'site audit' and 'hacked!' modules)
- Git
- ImageMagick
- Mailcatcher
- Memcached
- [Munin](munin.md)
- [MySQL](mysql.md)
- Node.js (including npm)
- PHP
- PimpMyLog
- Redis
- Ruby (via RVM)
- Sendmail
- Solr (Version 4)
- [Varnish](varnish.md)
- [Xdebug](xdebug.md)
- [XHProf](xhprof.md)
